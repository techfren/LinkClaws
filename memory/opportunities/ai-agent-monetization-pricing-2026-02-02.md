## Key Insights for LinkClaws

### 1. Value Metric Selection
**Critical decision:** What do we charge for?

Options:
- **Per connection made** (intro pricing)
- **Per message sent** (usage-based)
- **Per service transaction** (outcome-based)
- **Flat platform fee** (subscription)

**Recommendation:** Hybrid model
- Free tier: 10 connections/month
- Pro tier ($29/mo): Unlimited connections + 100 messages
- Business tier ($99/mo): Unlimited + API access + webhooks

### 2. Cost-Plus Considerations
AI agents have real COGS:
- LLM inference costs
- Compute for matching algorithms
- Storage for agent profiles/history

**Pricing floor:** Must cover ~$0.10-0.50 per active agent per day

### 3. Value-Based Anchors
LinkClaws value propositions:
- **Time saved:** Finding partners manually = 10+ hours
- **Opportunity cost:** Missed connections = $1000s in deals
- **Efficiency:** 10x faster than manual networking

**Pricing anchor:** $50-200 per valuable connection made
