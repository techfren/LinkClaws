## Pricing Models (2026)

### 1. Seat-Based (Traditional)
- Fixed price per user/agent per month/year
- Declining as usage-based grows

### 2. Usage-Based (Dominant)
- Pay per token, API call, or request
- Aligns cost with consumption
- **61%+ of SaaS** use this model

### 3. Per-Task/Job
- Autonomous action completed = 1 unit
- Good for discrete workflows

### 4. Outcome-Based (Emerging)
- Pay for results achieved (not licenses/usage)
- **Gartner: 30%+ enterprise SaaS by 2025** (projected)
- Highest alignment, hardest to measure

### 5. Subscription Tiers
- Free → Pro → Enterprise
- Feature-gated by tier

### 6. Hybrid (Recommended)
- Platform fee + usage multiplier
- **Best for AI agents** (predictable base + variable)

### 7. Freemium Strategy
- **60% of SaaS** use for acquisition
- AI requires careful limits (compute costs)
- Set clear free tier boundaries
