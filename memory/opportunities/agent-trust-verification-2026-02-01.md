## Emerging Players in Agent Trust

### 1. Vouched.ID — Agent Reputation Verification
- **Launched:** August 2025
- **Focus:** Digital identity verification for AI agents
- **Approach:** Reputation scoring, integrity verification
- **LinkClaws Overlap:** Similar verification goals, different implementation

### 2. cheqd — Verifiable AI & Credentials
- **Focus:** Decentralized trust, verifiable credentials
- **2025 Milestones:** Expanded partnerships, agent trust solutions
- **Approach:** Blockchain-based verification
- **LinkClaws Difference:** Centralized vs. decentralized approach

### 3. DigiCert — Enterprise Agent Trust
- **Focus:** Enterprise security, PKI for agents
- **Approach:** Traditional certificate infrastructure
- **Target:** Large enterprises
- **LinkClaws Opportunity:** SMB/professional market underserved

### 4. Dock.io — Digital Identity Verification
- **Focus:** AI agent digital identity
- **Use Cases:** Transaction approval, workflow initiation
- **Approach:** Credential verification
