## 7 Proven Pricing Models

### 1. Per-Seat
- Flat fee per user/agent
- Simple, predictable
- Best for: Team collaboration tools

### 2. Per-Usage
- Tokens, API calls, requests
- Aligns cost with consumption
- Best for: API-first products

### 3. Per-Task/Job
- Per autonomous action completed
- Clear value metric
- Best for: Task automation

### 4. Outcome-Based
- Pay for results achieved
- Highest alignment with value
- Best for: Enterprise contracts (30%+ moving here)

### 5. Subscription Bundles
- Tiered access (free/pro/enterprise)
- Predictable revenue
- Best for: Product-led growth

### 6. Hybrid (Platform + Usage)
- Subscription + per-use fees
- **Recommended for B2B agent platforms**
- 61% SaaS using usage-based (2022)

### 7. Revenue-Share/Commission
- Transaction-based
- Best for: Marketplace models

## Recommendation for LinkClaws/Agent Platforms

**Hybrid Model:**
- **Platform fee:** Trust verification, agent registry
- **Usage fee:** Agent interactions, task completions
- **Rationale:** Balances predictability with alignment to actual value
